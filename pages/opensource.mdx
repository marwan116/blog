# Open Source Projects

- [supreme-task](https://github.com/marwan116/supreme-task) A prefect extension that builds on top of the task decorator to reduce negative engineering!
- [model-composer](https://github.com/marwan116/model-composer) Easily compose a model from a set of base models
- [aws-parquet](https://github.com/marwan116/aws-parquet) A toolkit that provides an object-oriented interface for working with parquet datasets on AWS
- [pandas-chained-logging](https://github.com/marwan116/pandas-chained-logging) A logging accessor that goes hand in hand with pandas method chaining.
